# WhatsApp chat Analyzer
Description:

This project analyzes WhatsApp chat data using Python and Pandas.
The raw chat exported from WhatsApp is converted into a structured dataset containing date, time, user and message columns. The project performs basic data cleaning and analysis such as identifying the most active user and calculating message statistics.

Tools Used

Python

Pandas

NumPy

Features

Chat text parsing

Data cleaning

Media message handling

Message length analysis

User activity analysis
